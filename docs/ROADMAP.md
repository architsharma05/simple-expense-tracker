# Roadmap

## Phase 0: Foundation

- Preserve legacy Swing application
- Create monorepo structure
- Add Spring Boot backend scaffold
- Add Next.js frontend scaffold
- Add PostgreSQL schema migration
- Add project architecture docs

## Phase 1: Finance Platform MVP

- User registration and login
- JWT authentication
- Transaction CRUD
- Transaction search and filters
- Budget management
- Dashboard summary
- Category and monthly trend charts
- Swagger/OpenAPI documentation

## Phase 2: AI Copilot

- AI spending summary
- AI budget coach
- Natural language finance chat
- Stored insight history
- Prompt and data access safeguards

## Phase 3: Advanced AI

- Receipt upload and OCR
- Smart categorization
- Recurring expense detection
- Anomaly detection
- Spending forecasting
- Goal savings recommendations

## Phase 4: Production Readiness

- Unit tests
- Integration tests
- Dockerized app services
- CI/CD pipeline
- Rate limiting
- Audit logging
- Monitoring and error tracking
